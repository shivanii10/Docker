Creation of Project and Run:
- go to https://start.spring.io/; and create a maven project.
- Import the maven project in eclipse.
- In application.java define the mapping and endpoints.
- define the port in application.properties.. default port is 8080.
- run the application

web browser:
http://localhost:8084/
